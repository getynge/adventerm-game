We need an option to allow the user to configure things like keybinds and color schemes. This feature is concerned with two primary changes the addition of an options menu and the addition of color schemes. The options menu allows the user to configure things like keybinds and the color scheme that they currently have, Keybinds include the keys they use to move their character and interact with the world. Color schemes provide colors that make individual objects easier to identify. For example, the player And objects the player can interact with are brightly colored, ground the player can move on is lightly colored, and things the player is unable to interact with, like walls, have bland but visible coloring.
For now the specifics on color schemes do not matter, but it should be extremely easy to configure new color schemes - in principle the structs that store colorscheme information should be reasonably serializable into JSON format and creating new scheme should be a completely trivial task.
For obvious reasons user configuration should persist between runs
